Automatically assign numbers within blog posts.

== Description ==

This is a wordpress plugin developed for list blogs, which usually publish variety of lists articles like http://www.bloghug.com/project-management-softwares/ on their blog.

The plugin allows to automatically create numbers for h3 titles. Creating and editing numbers within posts can be sometime time consuming (especially when you want to re-order lists), this plugin automatically give number to all items which are titled under h3.

The plugin also allow to enable/disable numbered heating automatically within post. 

**Features**

* Automatically assign numbers (with matching blog style) to h3

* Enable/Disable this feature on individual post or page.

== Installation ==

* Uncompress the download package.

* Upload folder including all files and sub directories to the /wp-content/plugins/ directory.

* Activate the plugin through the 'Plugins' menu in WordPress.
